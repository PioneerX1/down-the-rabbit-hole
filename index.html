What's happening behind the curtain

Clicking on the button triggers a function, in this case, an explanation pops up like this one.

$("#element1-trigger").click(function() {SHOW SOME TEXT});

By clicking on the Fade In text, the output will now fade in instead of appearing all at once.

$("#element2-trigger").click(function() {$("element2-details").fadeIn();});

By clicking on the Toggle text, the output will toggle back and forth.

$("#element3-trigger").click(function() {$("element3-details").toggle();});

Shows the slideshow of the images that automatically slides.

$(".bxslider").bxSlider({ auto: true, autoControls: true });

By moving the mouse pointer over the targeted screen area, a variety of effects can be generated. In this case we used the .show() and .hide() effects.

$("#element5-trigger").mouseover(function(){$("#element5-details").show(); }); $("#element5-trigger").mouseout(function() {$("#element5-details").hide();});

< marquee> < /marquee> This is a cute feature that animates text using an HTML tag. Although this is considered an obsolete feature, we think that is a simple way to animate a few words for emphasis on a page.

$("#element6-trigger").click(function() {$("element6-details").toggle();});

The pageX property of the MouseEvent interface returns the X (horizontal) coordinate in pixels relative to the whole document. The pageY property of the MouseEvent interface returns the Y (vertical) coordinate in pixels relative to the whole document.

$("#element7-trigger").click(function() {$("element7-details").toggle();});

The mouseup event is fired when a pointing device button is released over an element. The mousedown event if fired when a pointing device button is released over an element.

$("p#element8-trigger").mousedown(function(){$("#mud").text('MOUSE CLICKED.');}); $("p#element8-trigger").mouseup(function() {$("#mud").text('MOUSE RELEASED.');});

The window screenX and screenY properties returns the x and y coordinates of the new window relative to the screen. We have paired this with a window.open() method to make the property easier to demonstrate.

$("#element8-trigger").click(function() {$("element9-details").toggle();});

This map is generated with a Google API that is populated with a key. It can be customized by selecting what kind of map you want. The initial or subsequent default location can be static or dynamic. This map is currently set to the Seattle Epicodus building location.
